Blocking URLs

To block a URL, follow these steps:

  • Go to the Options page.
  • Enter the URL or keyword you want to block in the input box.
  • Click the "Add" button to add the URL or keyword to the blocked list.

Types of Blocks

There are two types of blocks:

  • Website üåê: This option blocks specific websites. For example, to block example.com, it will be converted to a pattern that matches URLs like https://example.com.
  • Keyword üîç: This option blocks search queries containing specific keywords. For example, to block searches containing the word "example", it will be converted to a pattern that matches URLs like search?q=example or search_query=example.

Unblocking URLs

To unblock a URL or keyword, there are three ways:

  • From the Popup: Open the extension popup, find the URL or keyword in the list, and uncheck the checkbox next to it to disable it. Note that the URL or keyword needs to be starred to appear in the popup.
  • From the Options Page: Go to the Options page, find the URL or keyword in the list, and uncheck the checkbox next to it to disable it, or click the "Delete" button to remove it from the list entirely.
  • From the Blocked Page: When a website is blocked, you will see a blocked page. Click the "Unblock and Continue" button on this page to disable all URLs and keywords that caused the block. Upon doing so, you will be redirected to the page that was originally blocked.

Toggling the Blocker

You can enable or disable the blocker from the popup:

  • Click the "Enable Blocker" or "Disable Blocker" button to toggle the blocker status.
  • The current status of the blocker (ENABLED or DISABLED) will be displayed at the top of the popup.
