Overview

Website Blocker helps you limit distracting websites and search terms. It can block sites immediately, track block and unblock activity, temporarily unblock a site, redirect you toward useful alternatives, and show habit goals when you choose to remain focused.

The extension has three main areas: the popup, the blocked page, and the options page. The popup is for quick actions. The blocked page appears when a site is blocked. The options page is where you manage rules, settings, analytics, saved URLs, habits, and import/export.

Quick Start

To start blocking a website or search term:

  • Go to the Options page.
  • Open the Sites tab. If access is locked, complete the challenge in the Grant Access tab first.
  • Enter a website such as example.com or a search keyword such as games.
  • Click Add.
  • Leave the checkbox enabled for items you want actively blocked.

Grant Access

The Grant Access tab protects sensitive actions when hard mode is active. If one or more enabled items are in hard mode, you must solve math problems and then type a quote before opening the Sites tab or importing data.

Options, Analytics, Saved URLs, Habit Tracker, and Productive URLs remain available without the access challenge.

Sites

The Sites tab manages blocked websites and search terms.

  • Website rules match sites such as example.com.
  • Keyword rules match search query parameters such as q=example, s=example, or search_query=example.
  • Checkboxes turn individual rules on or off.
  • Stars add rules to the popup for faster access.
  • Easy/Hard controls how strict a rule is. Hard mode adds extra friction before disabling or unblocking.
  • Edit changes an existing rule.
  • Delete removes a rule.
  • Drag handles reorder rules.

Blocked Page

The blocked page appears when an enabled rule matches the page you are trying to visit. Depending on your settings, it can show block counts, unblock counts, a reason input, a confirmation step, a time selector, productive URLs, habit goals, and scripture verses.

  • Unblock Site disables the matching rule or temporarily unblocks it, depending on your settings.
  • Give Reason lets you record why you wanted to visit the site when reason input is enabled.
  • Remain Focused closes the tab, redirects to a saved page, displays your custom message, or shows habits and productive links.
  • Hard mode requires a quote challenge before temporary unblocking. Night mode can also prevent hard-mode unblocking between 9PM and 4AM.

Popup

The popup gives quick access to common actions.

  • Enable Blocker / Disable Blocker toggles the entire extension. If hard-mode items are enabled, disabling the blocker opens a challenge page first.
  • Options opens the options page.
  • Habit Tracker opens the Daily Goals section.
  • Help opens this page.
  • Starred site rules appear in the popup so you can quickly enable or disable them.
  • If the current tab is temporarily unblocked, the popup can show when it will be reblocked and provide a Reblock This Website button.

Blocking Options

The Options tab customizes how blocking and unblocking behave.

  • Blocked page background color changes the blocked page color.
  • Reason input asks why you are visiting a blocked page.
  • Disable unblock button delays the unblock button for a chosen number of seconds.
  • Time input asks how long a site should stay unblocked.
  • Confirm message adds a final confirmation before unblocking.
  • Temporary unblocking automatically reblocks a rule after the selected duration.
  • Night mode prevents hard-mode unblocking between 9PM and 4AM.
  • Save blocked URLs controls whether blocked URLs are saved always, only when a reason is given, or never.
  • When tab is closed chooses whether Remain Focused closes the tab or redirects to another page.
  • Display message shows a custom message after Remain Focused is pressed. You can also attach an optional link.
  • Reblocking notifications can notify you when a temporarily unblocked rule is reblocked.
  • Scripture verses can show verses on the blocked page and optionally require reading progress before unblocking.

Analytics and Saved URLs

The Analytics tab shows block and unblock counts by item and date. This can help you see which rules are triggered most often.

The Saved URLs tab lists blocked URLs that were saved according to your Save Blocked URLs setting. Saved entries include the URL, matching items, date, and reasons. Saved URLs older than 30 days are permanently deleted.

Habit Tracker

The Habit Tracker lets you define daily habits. Each habit can have an optional URL. When you choose Remain Focused on the blocked page, your habits can appear there. Closing that screen can send you to the first unfinished habit that has a URL.

  • Use Add to create a habit.
  • Check a habit when it is complete for the day.
  • Use Edit, Delete, and drag handles to maintain the list.
  • Daily completion resets automatically on a new local date.

Productive URLs

The Productive URLs tab manages links shown after choosing Remain Focused. Use it for study sites, task lists, documentation, courses, or any page that helps you redirect attention.

  • Productive URLs must start with https://.
  • Each URL can have a display name.
  • You can edit, delete, and reorder productive URLs.
  • Random display order can be enabled from this tab.

Import and Export

The Import/Export tab lets you back up or restore extension data. Export downloads a JSON file containing sync and local storage data. Import restores data from a previously exported file.

Import is protected by the Grant Access challenge when hard-mode items are active.

Privacy Notes

Website Blocker uses browser extension storage for rules, settings, analytics, saved URLs, reasons, habits, and productive links. Saved URLs and reasons can contain sensitive browsing information, so review the Saved URLs tab and exported JSON files before sharing them.

Use Delete All in Saved URLs to clear saved URL history. Saved URLs older than 30 days are also deleted automatically.
