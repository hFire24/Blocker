Website Blocker Help

Welcome to the Website Blocker extension! Here are the instructions on how to use it:

Blocking URLs

To block a URL, go to the Options page, enter the URL or a regular expression pattern in the input box, and click "Add URL". The URL will be added to the blocked list.

Unblocking URLs

To unblock a URL, uncheck the checkbox next to the URL on the Options page. The URL will be removed from the enabled list but will remain in the blocked list for future reference.

Testing Regular Expressions

You can test your regular expressions by going to the Testing page, entering a regex pattern and a URL, and clicking "Test". You will see a message indicating whether the URL matches the pattern.

Toggling the Blocker

You can enable or disable the blocker from the popup. Click the "Enable Blocker" or "Disable Blocker" button to toggle the blocker status.

Options Page

From the popup, you can open the Options page to manage your blocked URLs.

Examples

Here are some examples of regex patterns you can use:

  • Block the website example.com: ^https?://+([^:/]+\\.)?example\\.com[:/]
  • Block all search queries containing the word "example": .*example.*
